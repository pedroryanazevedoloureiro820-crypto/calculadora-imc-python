# Calculadora de IMC com Feedback Visual 

Este é um script em **Python 3** que calcula o Índice de Massa Corporal (IMC) e fornece um diagnóstico baseado nas faixas de peso da OMS. O diferencial deste projeto é o uso de cores no terminal para destacar o nível de atenção necessário para cada resultado.

## Funcionalidades
- **Entrada de dados:** Recebe altura e peso com suporte a números decimais.
- **Cálculo Automático:** Aplica a fórmula padrão do IMC: $IMC = \frac{peso}{altura^2}$.

## Lógica de Programação
O script utiliza estruturas condicionais (`if`, `elif`, `else`) para categorizar o resultado do cálculo. Também demonstra o uso de **f-strings** ou o método `.format()` para manipulação de strings e estilos no terminal.

## Como executar
1. Certifique-se de ter o Python 3 instalado.
2. Baixe o arquivo `imc-python.py`.
3. Rode o comando: `imc-python.py`
